<h1>Portfolio.</h1>

This is my portfolio.

<h1>Description</h1>
My portfolio is a single page application website built with HTML, CSS and JavaScript. 
The page includes:

- My projects with links to repositories and live websites
  
- Information about me
  
- Cover Letter and CV
  

<h1>Built with</h1>

- HTML

- CSS

- JavaScript

<h1>Getting started</h1>

1. Clone the repo:
```bash
gh repo clone siljeloge/portfolio
```

2. Install the dependencies:
```
npm install
```

<h1>Running</h1>

To run the app, run the following commands:

```bash
npm run start
```
